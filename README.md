# Swarm — Homebrew Tap

A native macOS BitTorrent client. https://swarmtorrent.com

## Install

```sh
brew install --cask swarmtorrent/tap/swarm
```

That's it. Swarm keeps itself up to date after install (via Sparkle).

## Update / uninstall

```sh
brew upgrade --cask swarm            # (Swarm also auto-updates itself)
brew uninstall --cask swarm          # remove the app
brew uninstall --zap --cask swarm    # remove the app and its data
```
